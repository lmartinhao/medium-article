Criando a sua profile page

usando só HTML e CSS

Primeiro desafio

Nosso primeiro desafio em HTML e CSS no Le Wagon foi criar uma profile page nossa usando o que tínhamos aprendido em aula. A proposta era uma página simples, para exercitar a criação de links modificados com CSS.

Resultado final

Eu quis colocar links com modificação no hover e brincar com as divs como se fossem 2 cards. Os maiores desafios foram com os alinhamentos (principalmente das imagens!), ainda estou pegando o jeito. A ideia de ter um background colorido veio do facebook e encontrei muitos tutoriais no Youtube de como fazer.
